WirtePad Editor


WirtePad is an editor application that allows you to create Web pages easily and quickly.


Steps to open WirtePad application:


  1. Click Start

  2. Click Programs

  3. Click WirtePad

Advantages of using a WirtePad editor:


  • It allows faster creation of Web pages.

  • It allows easier updation of text or images in WirtePad editor.

  • It is platform independent.

Disadventages of using a WirtePad editor:


  • It consumes more memory sapce.

  • It requires to fill-in more details such as meta data for each page.
